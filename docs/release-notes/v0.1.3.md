# v0.1.3

## 重点变化

- Windows 安装包改进了覆盖安装流程：安装新版本前会尝试根据系统登记的静默卸载命令先卸载旧版本，减少旧安装残留对新版本的影响。
- Windows 安装包继续强化用户 `PATH` 清理逻辑，降低重装后命令入口指向旧目录或留下失效路径的风险。

## 修复

- 修复 Windows 重装时旧安装目录可能继续残留在当前用户 `PATH` 中的问题。
- 修复 Windows 安装器执行旧版本静默卸载命令时的解析和执行方式，避免路径中包含空格或带参数的卸载命令被错误处理。
- 修正 Windows 安装日志中旧版本安装位置提示的格式。

## 变更

- Windows 安装器会检查 HKLM、WOW6432Node 和 HKCU 下的旧版本卸载信息，并在检测到可用的 `QuietUninstallString` 后先执行旧版本静默卸载。
- 项目版本从 `0.1.2` 升级为 `0.1.3`。

## 安装包

- Windows: `city-tier-stats-0.1.3-setup.exe`
- macOS: `city-tier-stats-0.1.3.pkg`
