# v0.1.0

## 重点变化

- 首次发布城市分层统计工具，支持从 Excel 或 CSV 文件读取“归属地”列，统计新一线、一线、二线、三线、四线和其他城市的数量及占比。
- 提供逐行明细导出能力，方便核对每条记录提取到的城市、归一化城市名和匹配到的城市分层。
- 提供 Windows 和 macOS 安装包构建流程，便于在桌面环境中安装和分发。

## 新增

- 支持读取 `.xlsx` 和 `.csv` 输入文件。
- 默认从 `归属地` 列提取城市，也可以通过 `--column` 指定其他列名。
- 支持中文地区字段中的常见连接符，并从类似 `省份-城市-区县` 的格式中提取城市部分。
- 支持城市名归一化匹配，例如去除末尾的“市”，并兼容“地区”“盟”“自治州”等常见后缀。
- 支持通过 `city_tiers.yaml` 自定义城市分层配置。
- 支持通过 `--detail-output` 导出 CSV 或 Excel 格式的逐行匹配明细。
- Windows 安装包会添加右键菜单入口，并把安装目录写入当前用户的 `PATH`。
- macOS 安装包会安装命令行入口，并尝试为当前登录用户安装 Finder Quick Action。

## 修复

- 卸载 macOS 安装包时，会尝试删除当前用户的 Finder Quick Action、命令入口、安装目录和安装记录。
- Windows 卸载流程会清理安装时加入当前用户 `PATH` 的目录。

## 变更

- 项目增加 pytest 单元测试，覆盖城市提取、归一化、CSV/XLSX 读取、YAML 配置校验、统计结果和明细导出。

## 安装包

- Windows: `city-tier-stats-0.1.0-setup.exe`
- macOS: `city-tier-stats-0.1.0.pkg`
